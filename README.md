Este proyecto es un dashboard de emogis, para poder practicar el uso de TailwindCSS y Typescript y el manejo de las clases de TailwindCSS con Typescript.

# 🚀 Instalación
```bash
npm install
```
# 🛠️ Uso
```bash
npm run dev
```
# 📦 Compilar
```bash
npm run build
```
